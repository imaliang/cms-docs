# VPS推荐

> [!IMPORTANT]
> 在不是被坑的情况下，一分价钱一分货

## 国内外VPS对比

| 项目 | 国内 | 国外 |
| --- | --- | --- |
| 带宽 | 3mbps (较低) | 1000mbps (较高) |
| 延迟 | 10ms (较低) | 150ms+ (较高) |
| 域名访问 | 需要备案 | 不需要备案 |
| 网络环境 | 部署一些国外服务时需要自己加代理 | 访问国外服务时不需要加代理，但是有的国内产品国外访问不友好 |

> [!TIP]
> 1. 注册时，最好关闭代理
> 2. 资料填写时，邮编和地址最好和本地IP匹配
> 3. 地区选择CN-China
> 4. 电子邮箱写自己的
> 5. 电话号码写中国的+86
> 6. 下单完成后你会收到一些邮件，其中有个邮件是激活验证邮件，一定要选择支付验证
> 7. 支付最好用paypal，退款时可以发工单让其原路返回
